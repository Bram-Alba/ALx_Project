# Daily Task Planner

The Daily Task Planner is a React-based web application that helps users manage their daily tasks efficiently.

## Features

- Add new tasks
- View task list
- Mark tasks as completed
- Delete tasks
- Deadline timer and notification system
- Data persistence using localStorage
- Filter tasks (All / Active / Completed)
- Timer for each task
- Progress bar showing task time

## Project Structure
src/components/
- Header.jsx
- TaskForm.jsx
- TaskList.jsx
- TaskItem.jsx
- App.jsx

## Built With

- React
- Vite
