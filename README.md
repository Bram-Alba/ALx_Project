# Daily Task Planner

The Daily Task Planner is a React-based web application that helps users manage their daily tasks efficiently.

## Features

- Add new tasks
- View task list
- Mark tasks as completed
- Delete tasks
- Deadline timer and notification system
- Data persistence using localStorage

## Project Structure

- App.jsx
- Header component
- TaskForm component
- TaskList component
- TaskItem component

## Built With

- React
- Vite
- JavaScript
- CSS
